# Task 6 - Sales Trend Analysis Using SQL

## Objective
Analyze monthly revenue and order volume using SQL aggregation functions.

## Tools Used
- MySQL Workbench
- SQL

## Dataset
Sample Superstore Dataset

## Concepts Used
- SELECT
- GROUP BY
- ORDER BY
- SUM()
- COUNT()
- COUNT(DISTINCT)
- Aggregate Functions
- STR_TO_DATE()
- YEAR()

## Analysis Performed

Calculated total sales revenue generated each month.

### Monthly Order Volume Analysis
Calculated total number of unique orders placed each month.

### Revenue and Order Analysis
Combined revenue and order volume to analyze monthly performance.

### Top 3 Months by Revenue
Identified the highest revenue-generating months.

## Key Insights

- Monthly order volume was analyzed.
- Top-performing months were determined.
- SQL aggregation functions were used for business analysis.
